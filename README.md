A Todo List web application

**How to run the app**
- Clone the repository
- Open in any IDE or editor
- Run **pip install -m requirements.txt**
- Run **flask run**
- Open the URL in the output terminal, it is usually : **http://127.0.0.1:5000/**
- Use
